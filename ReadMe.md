## Synopsis

This is repository for ITU Ride Share project “Share on Wheels”.

## Technologies used

**Frontend:**

* HTML
* CSS
* Java Script
* Angular JS framework


**Backend:**

* JAVA
* Spring context for dependency management
* RESTful services using Spring controllers
* Spring JDBC


**Database:**

* MySQL server

**Build Management**

* Maven
